Modèle de livre, rapport, thèse

3 Modèle d’ouvrage

Ce modèle s’appuie sur Bookdown pour permettre d’écrire en R Markdown un livre, un rapport, une thèse de doctorat ou un mémoire d’HDR.

Le fichier principal est index.Rmd qui contient la description de l’ouvrage dans son entête. Tous les autres fichiers .Rmd du dossier contiennent un chapitre. Le fichier MyBook.bib contient la bibliographie.

Ce fichier devra être supprimé, comme 81-Demarrage.Rmd et 82-Syntaxe.Rmd : ils doivent être remplacés par le contenu de l’ouvrage.

Pour commencer, créer un nouveau projet R à partir de ce dossier. Ouvrir ensuite index.Rmd et cliquer sur le bouton Build Book dans la fenêtre Build de Rstudio.

Attention, évolutions le 11/04/2019. Deux fichiers sont à mettre à jour dans les projets existants:

  • Bug dans bookdown qui ne prend plus en compte les déclarations de langues “otherlangs”. Le code pour contourner le bug est dans latex/preamble.tex, à supprimer à terme.

  • Mise à jour du modèle latex/pdgUniv.sty après une évolution de LaTeX.
